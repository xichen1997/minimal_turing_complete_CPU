

// language design:

Basic usage:
let a = 5; // assign value.
let a = b; // assign value from other variable.
let a = 5 + 7; // left associate calculation.
out x; // output the value, put the value in the 0xFF00 position.
out 10; // put 10 in the 0xFF00 position.
loop: // label, virtual parser node, will translated to the next location.
goto: // JMP to the label memory location.
if x <= y goto <label>: // if x <= y then go the <label>
